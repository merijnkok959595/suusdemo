OpenAI Realtime API
Pros
  • All-in-one: STT + LLM + TTS bundled
  • Mini model is the cheapest overall option
  • Single API, no service stitching
  • GPT-4o LLM quality baked in
  • Very low latency (single round-trip)
Cons
  • Fully locked into OpenAI
  • Full model (gpt-4o) is expensive
  • Limited voice/persona choices
  • Cannot swap to a cheaper LLM
Deepgram
Pros
  • Industry-leading STT accuracy
  • Cheapest STT at $0.0077/min
  • Bring your own LLM (any model)
  • $200 free credits, no expiry
  • Voice Agent API = simple setup
Cons
  • Voice Agent is $4.50/hr (priciest option)
  • Aura TTS quality is average
  • DIY pipeline needs more plumbing
  • Growth plan requires $4K+/yr commitment
ElevenLabs
Pros
  • Best-in-class TTS voice naturalness
  • Ultra-low TTS latency (~75ms Flash)
  • Voice cloning & custom personas
  • 90+ languages, STT + TTS together
  • Great for premium branded UX
Cons
  • TTS character costs add up fast
  • Need separate LLM (more complexity)
  • Most expensive for full pipeline
  • Not built for real-time voice primarily
// Bottom line recommendations

Keep OpenAI Realtime if...

You want the simplest stack at the lowest cost. The mini model at ~$1.08/hr is hard to beat. You're already integrated — don't over-engineer it.

Switch to Deepgram DIY if...

You want to swap your LLM (e.g., Claude, Llama) or need the most accurate transcription. Build: Deepgram STT + any LLM + Aura-2 TTS ≈ $2.30/hr.

Add ElevenLabs TTS if...

Voice quality is a product differentiator for you. Use ElevenLabs Flash for TTS only, keep Deepgram for STT, and pick your LLM. ~$3.18/hr but premium feel.

* Assumptions: 750 chars/min speech output, 600 audio tokens/min input+output, GPT-4o-mini LLM negligible at ~$0.001/min.
* OpenAI Realtime full (gpt-4o) ≈ $3.46/hr · Deepgram Voice Agent API ≈ $4.50/hr flat.
* Prices as of April 2026. Always verify at platform.openai.com, deepgram.com/pricing, elevenlabs.io/pricing/api.
